# Ray's Gemini CLI Skills

這是一個專為 Gemini CLI 設計的個人技能倉庫。

## 快速開始 (Quick Start)

### 1. Clone 倉庫

git clone https://github.com/ray-idempiere/ray-gemini-skills.git

### 2. 連結至 Gemini CLI

gemini skills link ~/sources/ray-gemini-skills

### 3. 驗證安裝

/skills list

--- 

## 包含技能

### 1. iDempiere Table Creator (idempiere-table-creator)

將業務表單（Word/Excel）自動轉換為符合 iDempiere 12 規範的資料庫表格 SQL DDL 及 AD_Element 同步語句。
